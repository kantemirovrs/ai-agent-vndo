# ai-agent-vndo
ИИ агент для работы с ВНДО
Краткое введение
Создание ИИ-агента для быстрого и удобного поиска релевантной информации в ИС "Внутренние нормативные документы организации" с использованием RAG. Задача быстрого, удобного поиска конкретизированной, оъективной информации в многочисленной, слабоструктурированной системе ВНДО. С обязательным указанием источника: наименования ВНД, его реквизитов, статьи, параграфа и т.д.

Контекст и проблематика
В силу огромного количества ВНДО, поиск необходимой информации занимает много времени. Наличие задач требующих сумаризации данных нескольких ВНД, большое количество их редакций, отсутствие ВНД в СПС «КонсультантПлюс» - усугубляют проблему. Поэтому, высока вероятность использования пользователем не релевантной информации, и, как следствие, его ошибочных действий. Кроме того, сотрудники без опыта, либо принятые не давно, могут не иметь достаточных знаний о структуре ВНД, месте из расположения, времени для ознакомления со всей базой ВНДО, так же могут испытывать большие сложности с поиском информации.

Данные
Данные представлены внутренними нормативными документами организации ВНДО, их объём может составлять порядка 1000-2000 шт. Как правило в формате pdf.

Методология
ИИ-агент с технологиями RAG , векторная БД

Результаты
Пока результаты отсутствую.

Планы на будущее
Расширение функционала ИИ-агента, например для формирования (на основании подобранной информации) документов отвечающих требования делопроизводства и корпоративного комплаенса.
